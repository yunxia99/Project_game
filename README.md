# Project_game
https://github.com/yunxia99/Project_game/edit/master/README.md
游戏的主要功能有通过作弊码实现自由选择关卡以及自由激活，取消回退功能；
作弊码使用方法：在通关界面或者在游戏中按ESC，回车，输入作弊码，再次回车；
选择关卡作弊码：root；
